<h1 align="center">📊 Weekly Progress Tracker</h1>

<p align="center">
🚀 Java Desktop Application | Task Management | AWT → Swing | File I/O → SQLite
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
The <b>Weekly Progress Tracker</b> is a Java-based desktop application designed to help users manage and track their weekly tasks in a structured and efficient way.
</p>

<p>
This project demonstrates a strong evolution in development approach — starting from a <b>basic AWT + File Handling system</b> and progressing into a more advanced <b>Swing-based application with SQLite database integration</b>.
</p>

<p>
It is not just a simple task manager, but a <b>complete productivity tracking system</b> that allows users to organize tasks, monitor progress, and analyze performance across different categories.
</p>

<hr>

<h2>🧠 Core Concept</h2>

<p>
The application is built around two main entities:
</p>

<ul>
  <li>📌 <b>Task:</b> Represents individual work with progress tracking (0% → 100%)</li>
  <li>📂 <b>Category:</b> Groups multiple tasks for better organization</li>
</ul>

<p>
Each task dynamically updates its status (<b>Not Started, In Progress, Completed</b>) based on progress values.
</p>

<hr>

<h2>🔥 Key Features</h2>

<ul>
  <li>✅ Create, Update, and Delete Tasks</li>
  <li>📂 Category-based task organization</li>
  <li>📊 Real-time progress tracking (0–100%)</li>
  <li>📈 Automatic status updates (based on progress)</li>
  <li>📉 Category-wise average progress calculation</li>
  <li>🧾 Data persistence (File + Database)</li>
  <li>📤 Export progress data to CSV</li>
</ul>

<hr>

<h2>🏗️ Dual Architecture (Important Highlight)</h2>

<p>
This project is built in <b>two versions</b>, showing a clear progression in development skills:
</p>

<h3>1️⃣ Basic Version (AWT + File I/O)</h3>

<ul>
  <li>🖥️ Built using Java AWT</li>
  <li>📁 Stores data in <b>weekly_progress_data.txt</b></li>
  <li>🧩 Uses arrays for managing categories and tasks</li>
  <li>🪟 Multi-window UI using dialogs</li>
</ul>

<h3>2️⃣ Advanced Version (Swing + SQLite)</h3>

<ul>
  <li>🎨 Built using Java Swing (modern UI)</li>
  <li>💾 Uses <b>SQLite database (tracker.db)</b></li>
  <li>📊 Displays data using JTable</li>
  <li>📉 Visual progress bars inside UI</li>
  <li>📤 Export feature for CSV reports</li>
</ul>

<p>
This transformation highlights the shift from <b>basic programming concepts to real-world application development</b>.
</p>

<hr>

<h2>⚙️ How It Works</h2>

<ul>
  <li>📌 User creates categories (e.g., Study, Fitness, Projects)</li>
  <li>📌 Tasks are added under each category</li>
  <li>📌 Progress is updated manually or calculated using parts</li>
  <li>📌 System automatically updates task status</li>
  <li>📌 Data is stored and retrieved from file or database</li>
</ul>
